## セットアップ

```bash
npm install
```

## 利用可能なスクリプト

- `npm run dev` — Vite 開発サーバーを起動
- `npm run lint` — ESLint による静的解析
- `npm run test:unit` — Vitest でユニットテストを実行
- `npm run test:e2e` — Playwright でE2Eテストを実行
- `npm run build` — 型チェックと本番ビルドを実行
